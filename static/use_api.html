Game Map Analyzer
Game Map Analyzer是什麼？
Game map analyzer是一個分析遊玩影片的API。
利用Machine Learning技術進行物件辨識，再進行物件之間的相對距離計算。
上傳一個影片，您將可以得到以下資訊，作為遊玩的攻略參考！
  • 關卡的全景圖
  • 移動與跳躍的軌跡
  • 物件辨識的影片
如何使用Game Map Analyzer？
請先免費加入會員，即可取得API key進行影片解析。
API docs:
使用方法目前有兩種：
  1. 使用者錄製並上傳遊戲影片後，使用網站介面或API送出申請。
  2. 使用者上傳遊戲影片至Youtube後，使用網站介面或API送出申請，由伺服器下載影片進行解析(Youtube改版，目前此功能維護中)
為確保服務品質，測試期間開放長度小於90秒的Youtube影片。
