Game Map Analyzer
如何使用GMA API：
請先加入會員後，即可取得API key進行影片解析。
請使用Youtube Video ID，以POST method送出request到以下URL: http://traces.fun/api/process-video/
並傳入body: {"youtubeid":(需要下載的video id), "token":(您的API token)}
伺服器會下載您的影片進行解析。
為確保服務品質，測試期間開放長度小於1分鐘的Youtube影片。請確保您的Youtube影片為公開。
